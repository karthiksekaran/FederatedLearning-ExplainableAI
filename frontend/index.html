About FedMedAI

Federated Learning

Implements federated learning using Flower framework. Multiple healthcare centers collaboratively train machine learning models without sharing sensitive patient data.

Dataset

Uses the UCI Liver Patient Dataset (ILPD) for binary classification of liver disease with biochemical markers and patient demographics.

Explainable AI

Uses SHAP (SHapley Additive exPlanations) to provide transparent insights into model predictions.

LLM Integration

Google Gemini provides natural language clinical interpretations for healthcare professionals.

Technology Stack

Flower PyTorch SHAP Google Gemini Flask Tailwind CSS
